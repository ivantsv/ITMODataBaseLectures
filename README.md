    1. –•–µ—à–∏—Ä–æ–≤–∞–Ω–∏–µ–º (+ –ø–∞—Ä–∞–ª–ª–µ–ª—å–Ω–æ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ —Ö–µ—à–∏—Ä–æ–≤–∞–Ω–∏–µ–º ‚Äî —à–∞—Ä–¥—ã).
    2. –°–ª–∏—è–Ω–∏–µ–º (–∫–ª—é—á–∏ –æ—Ç—Å–æ—Ä—Ç–∏—Ä–æ–≤–∞–Ω—ã / —Å–æ—Ä—Ç–∏—Ä—É–µ–º —á–µ—Ä–µ–∑ Quicksort –≤ PG).
    3. –í–ª–æ–∂–µ–Ω–Ω—ã–º —Ü–∏–∫–ª–æ–º (–≤–Ω–µ—à–Ω—è—è ‚Äî –º–∞–ª–µ–Ω—å–∫–∞—è, —É –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–π ‚Äî –∏–Ω–¥–µ–∫—Å).
*   **–°–µ–ª–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å:** –í—ã—Å–æ–∫–∞—è (–º–∞–ª–æ —Å—Ç—Ä–æ–∫), –ù–∏–∑–∫–∞—è (–º–Ω–æ–≥–æ —Å—Ç—Ä–æ–∫).
*   **–ú–µ—Ç–æ–¥—ã –¥–æ—Å—Ç—É–ø–∞:**
    1. –ü–æ—Å–ª. —Å–∫–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ (`Seq Scan`) $\rightarrow$ –ù–∏–∑–∫–∞—è —Å–µ–ª–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å.
    2. –ò–Ω–¥–µ–∫—Å—ã $\rightarrow$ –í—ã—Å–æ–∫–∞—è —Å–µ–ª–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å.
    3. –ì–∏–±—Ä–∏–¥ (–ò–Ω–¥–µ–∫—Å $\rightarrow$ Bitmap $\rightarrow$ –ü–æ—Å–ª.) $\rightarrow$ –°—Ä–µ–¥–Ω—è—è —Å–µ–ª–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å.
*   **Gather + Gather Merge** (–ü–∞—Ä–∞–ª–ª–µ–ª–∏–∑–º). –û–ø—Ç–∏–º–∏–∑–∞—Ç–æ—Ä —Å—á–∏—Ç–∞–µ—Ç —Å—Ç–æ–∏–º–æ—Å—Ç—å.

## 9. –ë–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç—å –∏ –ü—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ
*   **–ë–µ–∑–æ–ø–∞—Å–Ω–∞—è —Å–∏—Å—Ç–µ–º–∞:** –ó–∞—â–∏—â–µ–Ω–∞ –æ—Ç –Ω–µ—Å–∞–Ω–∫—Ü–∏–æ–Ω–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –¥–æ—Å—Ç—É–ø–∞.
*   **CIA Triad:** Confidentiality (–∫–æ–Ω—Ñ–∏–¥.), Integrity (—Ü–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç—å), Availability (–¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç—å).
*   **–ü—Ä–æ—Ü–µ—Å—Å—ã:** –ò–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏—è (–∫—Ç–æ —ç—Ç–æ?) $\rightarrow$ –ê—É—Ç–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ü–∏—è (–¥–æ–∫–∞–∂–∏! –≤—Ö–æ–¥) $\rightarrow$ –ê–≤—Ç–æ—Ä–∏–∑–∞—Ü–∏—è (—á—Ç–æ –º–æ–∂–Ω–æ? –ø—Ä–∞–≤–∞).
*   **–ú–æ–¥–µ–ª–∏ –¥–æ—Å—Ç—É–ø–∞:**
    1. **DAC (–î–∏—Å–∫—Ä–µ—Ü–∏–æ–Ω–Ω–∞—è):** –í–ª–∞–¥–µ–ª–µ—Ü —Å–∞–º —Ä–µ—à–∞–µ—Ç –∫–æ–º—É –∫–∞–∫–∏–µ –ø—Ä–∞–≤–∞. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç ACL.
    2. **MAC (–ú–∞–Ω–¥–∞—Ç–Ω–∞—è):** –¶–µ–Ω—Ç—Ä–∞–ª—å–Ω–∞—è –ø–æ–ª–∏—Ç–∏–∫–∞, –≤–ª–∞–¥–µ–ª–µ—Ü –Ω–∏—á–µ–≥–æ –Ω–µ —Ä–µ—à–∞–µ—Ç. –°–≤–µ—Ä—Ö–±–µ–∑–æ–ø–∞—Å–Ω—ã–µ —Å–∏—Å—Ç–µ–º—ã. –ù–µ–ª—å–∑—è –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –ø—Ä–∞–≤–∞.
*   **PostgreSQL:** –ü—Ä–∏–≤–∏–ª–µ–≥–∏–∏ (–Ω–∞ –ë–î, —Å—Ö–µ–º–µ, —Ç–∞–±–ª–∏—Ü–µ, –∫–æ–ª–æ–Ω–∫–∞—Ö), –†–æ–ª–∏ (users, –≥—Ä—É–ø–ø—ã, –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –ø—Ä–∞–≤–∞), –ù–∞—Å–ª–µ–¥–æ–≤–∞–Ω–∏–µ –ø—Ä–∏–≤–∏–ª–µ–≥–∏–π.
*   **–§—É–Ω–∫—Ü–∏–∏ / –ü—Ä–æ—Ü–µ–¥—É—Ä—ã / –¢—Ä–∏–≥–≥–µ—Ä—ã:**
    *   –§—É–Ω–∫—Ü–∏–∏: —á—Ç–æ-—Ç–æ –≤–æ–∑–≤—Ä., `SELECT`. –í–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å: `IMMUTABLE`, `STABLE`, `VOLATILE`.
    *   –ü—Ä–æ—Ü–µ–¥—É—Ä—ã: —É–ø—Ä–∞–≤–ª—è—é—Ç —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏.
    *   –¢—Ä–∏–≥–≥–µ—Ä—ã: –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è —Ä–µ–∞–∫—Ü–∏—è –Ω–∞ `INSERT`/`UPDATE`/`DELETE`.

## 10. –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ –ë–î
*   **–ì–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω–æ–µ –º–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞–Ω–∏–µ:**
    *   **–†–µ–ø–ª–∏–∫–∞—Ü–∏—è:** (–ö–æ–ø–∏–∏). –ê—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã: Master-Slave, Master-Master, Peer-to-Peer. –°–∏–Ω—Ö—Ä–æ–Ω–Ω–∞—è / –ê—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–∞—è.
    *   **–®–∞—Ä–¥–∏–Ω–≥:** (–†–∞–∑–¥–µ–ª–µ–Ω–∏–µ). –°—Ç—Ä–∞—Ç–µ–≥–∏–∏: –ü–æ –¥–∏–∞–ø–∞–∑–æ–Ω—É, –ü–æ —Ö–µ—à—É, –ü–æ —Å–ø—Ä–∞–≤–æ—á–Ω–∏–∫—É, –ì–µ–æ—à–∞—Ä–¥–∏–Ω–≥.
    *   **–ü–∞—Ä—Ç–∏—Ü–∏–æ–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ $\neq$ –®–∞—Ä–¥–∏—Ä–æ–≤–∞–Ω–∏–µ!**
*   **–ü—Ä–æ–±–ª–µ–º—ã —à–∞—Ä–¥–∏—Ä–æ–≤–∞–Ω–∏—è:** Join'—ã, –ê–≥—Ä–µ–≥–∞—Ç—ã, –†–µ—à–∞—Ä–¥–∏–Ω–≥, –ù—É–∂–µ–Ω –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç–æ—Ä.
*   **–¢–∏–ø—ã —Ä–∞—Å–ø—Ä–µ–¥. –°–£–ë–î:** –û–¥–Ω–æ—Ä–æ–¥–Ω—ã–µ / –ù–µ–æ–¥–Ω–æ—Ä–æ–¥–Ω—ã–µ.
*   **2PC (Two-phase commit):** –ë–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç—å –≤ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∫–µ.
*   **CAP (–ë—Ä—é–µ—Ä–∞):** CP (–∫–æ–Ω—Å–∏—Å—Ç–µ–Ω—Ç–Ω–æ—Å—Ç—å –ø—Ä–∏ —Ä–∞–∑—Ä—ã–≤–µ) vs AP (–¥–æ—Å—Ç—É–ø–Ω–æ—Å—Ç—å –ø—Ä–∏ —Ä–∞–∑—Ä—ã–≤–µ).
*   **PACELC:** P-A-C (–ø—Ä–∏ –∞–≤–∞—Ä–∏–∏) + E-L-C (Else ‚Äî –≤ —à—Ç–∞—Ç–Ω–æ–º —Ä–µ–∂–∏–º–µ: Latency vs Consistency).
*   **BASE:**
    *   **BA:** Basically Available (–≤—Å–µ–≥–¥–∞ –æ—Ç–≤–µ—á–∞–µ—Ç).
    *   **S:** Soft state (—Å–æ—Å—Ç–æ—è–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ –º–≥–Ω–æ–≤–µ–Ω–Ω–æ —Å–æ–≥–ª–∞—Å–æ–≤–∞–Ω–æ).
    *   **E:** Eventual consistency (—Å–æ –≤—Ä–µ–º–µ–Ω–µ–º —Å—Ç–∞–Ω–µ—Ç –∫–æ–Ω—Å–∏—Å—Ç–µ–Ω—Ç–Ω—ã–º).
 

*   **–£—Ç–æ—á–Ω–µ–Ω–∏–µ –ø–æ —Ñ–∞–Ω—Ç–æ–º–∞–º:**
    *   **Non-repeatable read (–ù–µ–ø–æ–≤—Ç–æ—Ä—è—é—â–µ–µ—Å—è —á—Ç–µ–Ω–∏–µ):** —ç—Ç–æ –∫–æ–≥–¥–∞ —Ç—ã –ø—Ä–æ—á–∏—Ç–∞–ª **—Å—Ç—Ä–æ–∫—É**, –∞ –∫—Ç–æ-—Ç–æ –µ—ë **–∏–∑–º–µ–Ω–∏–ª –∏–ª–∏ —É–¥–∞–ª–∏–ª**.
    *   **Phantom read (–§–∞–Ω—Ç–æ–º–Ω–æ–µ —á—Ç–µ–Ω–∏–µ):** —ç—Ç–æ –∫–æ–≥–¥–∞ —Ç—ã –ø—Ä–æ—á–∏—Ç–∞–ª **–Ω–∞–±–æ—Ä —Å—Ç—Ä–æ–∫** (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤—Å–µ—Ö —Å—Ç—É–¥–µ–Ω—Ç–æ–≤ —Å –æ—Ü–µ–Ω–∫–æ–π "5"), –∞ –∫—Ç–æ-—Ç–æ **–≤—Å—Ç–∞–≤–∏–ª (INSERT)** –Ω–æ–≤–æ–≥–æ —Å—Ç—É–¥–µ–Ω—Ç–∞. –í –∏—Ç–æ–≥–µ –ø—Ä–∏ –ø–æ–≤—Ç–æ—Ä–Ω–æ–º –∑–∞–ø—Ä–æ—Å–µ —É —Ç–µ–±—è –≤ —Å–ø–∏—Å–∫–µ –ø–æ—è–≤–∏–ª—Å—è ¬´–ø—Ä–∏–∑—Ä–∞–∫¬ª (–Ω–æ–≤–∞—è —Å—Ç—Ä–æ–∫–∞).
*   **–ü—Ä–æ —É—Ä–æ–≤–Ω–∏:** –ú–æ–ª–æ–¥–µ—Ü. –ú–∞–ª–µ–Ω—å–∫–∏–π ¬´–±–æ–Ω—É—Å¬ª: –≤ PostgreSQL —É—Ä–æ–≤–µ–Ω—å `Repeatable Read` **—É–∂–µ** –∑–∞—â–∏—â–∞–µ—Ç –æ—Ç —Ñ–∞–Ω—Ç–æ–º–æ–≤ (—Ö–æ—Ç—è –ø–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç—É SQL –Ω–µ –æ–±—è–∑–∞–Ω). `Serializable` –≤ Postgres ‚Äî —ç—Ç–æ –µ—â–µ –±–æ–ª–µ–µ –º–æ—â–Ω–∞—è —à—Ç—É–∫–∞ (SSI), –∫–æ—Ç–æ—Ä–∞—è —Å–ª–µ–¥–∏—Ç –∑–∞ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—è–º–∏ –º–µ–∂–¥—É —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏.

*   **–¢–µ—Ä–º–∏–Ω:** –¢–∞–∫–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞ –≤ –ø–∞–º—è—Ç–∏ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è **Dirty Page** (¬´–ì—Ä—è–∑–Ω–∞—è —Å—Ç—Ä–∞–Ω–∏—Ü–∞¬ª). –≠—Ç–æ —Å—Ç—Ä–∞–Ω–∏—Ü–∞, –¥–∞–Ω–Ω—ã–µ –≤ –∫–æ—Ç–æ—Ä–æ–π –≤ RAM –Ω–æ–≤–µ–µ, —á–µ–º –Ω–∞ –¥–∏—Å–∫–µ.
*   **–°–±–æ–π:** 
    *   **–°–µ–∫—Ä–µ—Ç –≤—ã–∂–∏–≤–∞–Ω–∏—è:** –°–£–ë–î —Å–Ω–∞—á–∞–ª–∞ –ø–∏—à–µ—Ç –∏–∑–º–µ–Ω–µ–Ω–∏–µ –≤ –ª–æ–≥ –Ω–∞ –¥–∏—Å–∫ (WAL), –∏ —Ç–æ–ª—å–∫–æ –ø–æ—Ç–æ–º –º–µ–Ω—è–µ—Ç —Å—Ç—Ä–∞–Ω–∏—Ü—É –≤ RAM (Shared Buffers). –õ–æ–≥ ‚Äî —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ –¥–ª–∏–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ ¬´—á—Ç–æ –º—ã —Å–¥–µ–ª–∞–ª–∏¬ª. 
    *   **–í–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏–µ:** –ö–æ–≥–¥–∞ —Å–≤–µ—Ç –≤–∫–ª—é—á–∞—Ç, –°–£–ë–î –ø—Ä–æ—á–∏—Ç–∞–µ—Ç WAL –∏ ¬´–ø—Ä–æ–∏–≥—Ä–∞–µ—Ç¬ª –≤—Å–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∑–∞–Ω–æ–≤–æ (**Redo**), –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–∏–≤ —Ç–µ –∏–∑–º–µ–Ω–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ —É—Å–ø–µ–ª–∏ –¥–æ–ª–µ—Ç–µ—Ç—å –¥–æ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Ñ–∞–π–ª–æ–≤ –¥–∞–Ω–Ω—ã—Ö.
